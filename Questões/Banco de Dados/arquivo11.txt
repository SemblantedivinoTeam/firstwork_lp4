O que é uma chave estrangeira em um banco de dados relacional?
Valor único que identifica unicamente uma tabela
Valor que indica a posição física dos dados no disco
Valor que indica a ordem dos registros na tabela
Valor que representa uma relação entre tabelas
Valor que define a restrição de integridade referencial

Uma chave estrangeira é um valor que estabelece uma relação entre tabelas, referenciando a chave primária de outra tabela.
